# Crypto Futures Signal Dashboard

A human-first crypto trading dashboard for memecoin futures.

## Features
- MEXC & Gate.io support
- Scalping + swing logic
- 3m / 5m / 15m / 1h / 4h timeframes
- HTF bias confirmation
- Plain-language trade reasoning
- Plain-language invalidation
- Mobile-friendly table view

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
